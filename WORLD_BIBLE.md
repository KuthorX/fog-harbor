# 雾港档案系列：世界观圣经

## 系列定位

每作是一份可独立通关的旧网络档案。玩家扮演不同时代的“恢复者”，通过过时的软件、地方网站、公共记录和私人通信追查一则都市传说。多数异常有科学或人为解释；每作结尾保留一项违反既有解释的小事实。

核心主题：人如何借技术保存记忆，又如何借集体叙事改写一个人。

## 不可改写规则

1. 雾港是虚构的中国沿海工业城市，1988 年设市，大学、广播台、港口和电子工业构成城市记忆。
2. 超自然从不被权威证实；理性解释必须覆盖主要事件，但不能抹去人物感受。
3. “ECHO”不是超自然组织，而是一套研究预期、记忆与传播的松散项目代号。
4. 每次 ECHO 相关档案恢复后，都会出现一个晚于封存时间的访问记录。来源长期不解释。
5. 邵雯拥有拒绝被叙事消费的权利。后续作品不得把她重新写成鬼、幕后恶人或单纯受害者。
6. 真实科学结论不因剧情需要而反转；未知来自样本、记录和人的局限。

## 关键机构

- 雾港大学：北楼是物理、心理和计算机兴趣小组共用实验楼。
- 雾港之声 AM1702：1999 年停播的地方中波电台，遗留大量听友信件和 QSL 卡。
- ECHO：1998—2000 年的认知暗示实验；后续发现其方法被商业和公共机构复用。
- 蓝鲸照相馆：地方影像档案节点，纸质底片比官方数据库保存得更久。
- 镜像室：校网维护组织。签名 S.W. 的值班记录与邵雯姓名形成可疑重合，但第一作不确认二者关系。

## 主时间线

- 1981：红星录像厅前身开业，后被附会成 POLYBIUS 黑色街机出现地。
- 1994：雾港之声开设听友点名节目。
- 1998：ECHO 项目立项。
- 1999-12-31：跨年实验、停电、北楼传说集中爆发。
- 2000-01-01：校园网镜像封存；邵雯离校。
- 2003：港区出现“收到未来短信”的传闻，短信中心时间校准记录缺失。
- 2008：蓝鲸照相馆关闭，一批底片流入市档案馆。
- 2012：社交网站出现使用 ECHO 术语的营销实验。
- 2026：第一份镜像被恢复。

## 系列规划

### 第一作《雾港在线 2000：零点调频》

局部谜团：北楼第八个人是否失踪。结论：认知实验和集体复述改写了邵雯主动离开的事实。

跨作伏笔：晚于封存日期的访问时间；镜像室签名 S.W.；ECHO 方法去向；零字节图片“第八张脸.bmp”。

### 第二作《雾港短信 2003：不存在的收件人》

媒介：小灵通、短信中心、地方论坛、运营商话单。科学核心：基站定位误差、短信延迟、时钟漂移。传说核心：收到三年后自己发来的短信。

回收：解释第一作部分 00:17 时间戳来自校时链路；发现 ECHO 被通信行为研究采购。

### 第三作《蓝鲸底片 2008：没有快门的照片》

媒介：照相馆库存、OCR、数码相机 EXIF、城市拆迁图。科学核心：双重曝光、滚动快门、面孔空想性错视。传说核心：每卷底片多出一张照片。

回收：零字节图片对应的原始底片存在，但拍摄日期早于 ECHO。

### 第四作《雾港热榜 2012：所有人都记得》

媒介：早期社交网站、营销后台、删帖记录、搜索趋势。科学核心：虚假记忆、信息级联、推荐偏差。传说核心：一夜之间全城记住不存在的电视节目。

回收：揭示跨作异常访问记录由一个自动归档代理生成，但代理训练数据仍无法解释。
