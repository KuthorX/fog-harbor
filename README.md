# 雾港在线 2000

原生 HTML/CSS/JavaScript 制作的千禧年都市传说网页 ARG。无需构建工具。

```bash
python3 -m http.server 4173
```

打开 <http://localhost:4173>。

在线试玩：<https://kuthorx.github.io/fog-harbor/>

独立联动《苍绿之眼：和弦夜班》：<https://kuthorx.github.io/fog-harbor/night-shift/>

联动现含十一个连续夜班案件与一次继承记录的复核周目。玩法使用千禧年内部桌面、分类器、版本挂载器、读带器、事故表、投射卡、光谱柜和十一轨混音台，不设单选问答。线索连接火爪、Husburd、7461-K、商徵、和弦能源、错误投射、大通禅师和凯特的未送达坐标；同编号角色、硬件与叙述声音仍按不同实体层处理。

游戏不显示进度和自动提示。建议玩家自己记录访问页面、关键词、密码与未解释的信息。

夜班答案合同自检：

```bash
node night-shift/verify.cjs
```

主作谜题合同自检：

```bash
node verify.cjs
```

内容包含：千年虫、午夜电台、连锁邮件、镜像延迟、街机都市传说、无线电传播、元素周期表、摩斯电码、视觉后像、选择性注意与记忆重构。
