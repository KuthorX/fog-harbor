# 《苍绿之眼：和弦夜班》联动圣经

## 定位

这是《苍绿之眼》与《雾港在线》的独立浏览器 ARG，不是角色客串包。

玩家是月球密室能源研究所的临时夜班复核员。和弦系统不断投递来自不同位面的谜题房间，提交答案会产生能源，但系统不说明能源来自哪里。某次夜班后，值班守则开始出现不同作者的改写。玩家必须判断：规则保护的是谁，规则牺牲的又是谁。

内容边界：微心理恐怖、规则矛盾、身份错置、时间线重复；无跳脸、血腥追逐和超自然定论。

## 原作事实与边界

以下内容可作为游戏硬规则：

1. 斗篷使用 K-1647 编号，是半碳基半硅基生命，从丛林中形成；先学呼吸、听觉，再学视觉和符号。
2. 斗篷会吸收时间，早期能长期保存符号；接触碳基生命会获得智识，夜间戴兜帽消化。
3. 斗篷初期会重复一句话最后的宾语。它不只是记录者，也会使阅读其文字的人被叙述感染。
4. 耳鱼 A-4597 能穿越时空，不能改变客观事实；大量重复经历让他的记忆与死亡次数难以线性描述。
5. 火爪 S-6458 寻找归宿，也寻找失踪的 Husburd。她是猫形生命，却不是红绿色盲。
6. 火爪的三把刀不是普通武器：撼地移动无生命物体；断/斩可切断物质，偶尔切断概念；Kantin 关联继续与希望。
7. 大通禅师 A-4249 曾是饭店老板，后以沙发形态存在；信奉商人三大守则和三条沟通原则。
8. 死象 D-XXXX 是被苍绿之眼注视后幻化成的地点。那里时空混乱；恐惧死亡者会看到红海与不停的雨。
9. 位面 A 存在错误投射：读者被投进方立的身份，获得西宫/X-3467之名；目标原本是凯特·沃尔。
10. 月球密室能源研究所使用“和弦”系统生成密室。研究员提交答案后产生能源，能源原理和去向对研究员保密。
11. 凯特·沃尔 S-6459 是普森·沃尔的女儿、火爪的朋友、克墓守护者。她不作为本作叙事主体，不描写其心理活动。
12. 原作计划包含位面 0、A、B、C；位面 A 约有 11 个案件，位面 B 才进入凯特琳·沃尔的最后一案。
13. 2342 年存在 K-1647 芯片：稀有、自学习、无中枢神经网络连接；S-6458 芯片较常见。
14. K-1647 机器人通常具有审视、情绪和电影视觉模块，也能自行改造或新增人格模块。
15. 机器人每天有十万分之二的随机自毁概率。单日概率很低，长期存活率会持续下降。
16. `7461-K` 是 `K-1647` 倒序，也是低频声波名称；该声波未造成人类实际死亡，却造成大量非 K-1647 机器人死亡。
17. 《五万分之一》的法庭层出现 S-6458 广播执行“机器人四大法律”，并出现复数 K-1647 观察者。
18. K-1647 也是音乐发布身份和跨作品叙述声音；不能据编号直接断定角色、芯片、机器人、作者投影是同一实体。

以下只能作为待证假说，游戏不能当成旁白事实：

- K-1647、斗篷、火爪、耳鱼是否等同于普森生命托付后的简单“三等分”。原文描述为意识糅合与分裂，但各角色已有独立历史。
- 苍绿之眼究竟是人物、观察行为、能力、读者位置还是 Grenyes 的接口。
- 和弦系统产生的能源是否来自被解决、被删除或被固定下来的故事线。
- 2442、2X25、2014、2025 等时间是否属于同一时间轴。
- 谁创建了投射系统，谁在利用错误坐标寻找凯特。
- 斗篷 K-1647、K-1647 型机器人、复数 K-1647 观察者、音乐发布者 K-1647 之间是否共享连续意识。
- 火爪 S-6458 与 S-6458 芯片、法庭广播执行者之间是同一存在、投射映射还是编号复用。

## 四层实体模型

任何同名编号先按四层分开记录，只有原作提供跨层行动证据时才连线：

| 层 | 例子 | 可验证内容 | 不可直接推出 |
|---|---|---|---|
| 角色层 | 斗篷、火爪、耳鱼、商徵、凯特 | 外貌、行为、关系、角色卡能力 | 等同于同编号硬件 |
| 型号层 | K-1647 芯片、S-6458 芯片、7461-K | 技术手册、概率、模块、适用法律 | 拥有角色层全部记忆 |
| 声音层 | 【审视】、〖情绪〗、/宗师/、法庭广播 | 说话格式、立场、可观测记录 | 声音背后只有一个主体 |
| 作者投影层 | F、KuthorX、七点半、K-1647 发布者 | 创作、音乐、博客、与普森对话 | 作者生平等于虚构年表 |

规则怪谈的状态错误来自跨层误识别。例如系统检测到 `S-6458` 后套用机器人四大法律，但当前房间中的 S-6458 可能是火爪；玩家必须找出系统用了哪一层证据。

## 五条故事线

### L0：斗篷的感觉与符号

呼吸 -> 两千种声音 -> 视觉 -> 符号 -> 酒馆学习宾语 -> 阅读者受感染。

玩法职责：修改界面语法、代词、按钮宾语。玩家选择一句规则时，可能因此成为规则里的角色。

### L1：酒馆、火爪、耳鱼与死象

斗篷进入酒馆 -> 火爪用撼地测试 -> 酒馆升起 -> 火爪与耳鱼回忆重复时间线 -> 刀被交给斗篷 -> 死象落地。

玩法职责：物件位置、空间规则、死亡重复、无法斩断的概念。

### LA：错误投射与西宫

玩家被投进2014年的方立 -> 声音承认坐标错误 -> 玩家获得西宫/X-3467身份 -> 与尚未成为叙事主体的凯特同行寻找黑匣子。

玩法职责：身份不是问卷选项，而是连续行动的结果。系统称呼随选择变化。

### LM：月球和弦研究所

商徵入职 -> 绩效要求迫使研究员解密室 -> S级案件复现真实谋杀 -> 提交答案产生能源 -> 投递坐标指向凯特。

玩法职责：夜班工作台、规则手册、案件提交、能源伦理，是本作主舞台。

### LP：普森与最后一案

普森的案件与死亡 -> 生命通过死象托付 -> 凯特承受父亲身份带来的注视 -> 位面 A 寻找凯特 -> 位面 B 才允许最后一案真正开始。

玩法职责：本作只留下父女关系的外部记录，不替凯特解释感受，不提前消费位面 B 的答案。

### LR：机器人、法庭与五万分之一

2342 年 K-1647 芯片 -> 每日随机自毁 -> 7461-K 低频声波危机 -> 人类被告自称机器人 -> 地下实验室发现 K-1647 -> 多作者手稿与法庭重构 -> S-6458 广播执行法律 -> K-1647 终止帷幕。

玩法职责：把“规则是否适用”置于“规则写了什么”之前。机器人法律保护整体人类时可能合法杀死个体；声波安全公告保护人类却忽略大量机器人死亡。

### LF：作者、父亲、普森与星网

早期普森探案 -> F 让普森阅读并推理后来的作品 -> 病床上的普森承认自己不等于文字角色 -> 《漫长告别》用父亲、普森和星网共同记录无法复述的现实 -> 商徵同时连接普森朋友、七点半故人和 KuthorX。

玩法职责：记录不能复活现场，也不能替当事人下结论；但多人共同保存、补写和回应，可以抵抗单一版本吞没其他人的经历。

### LMU：音乐作品

《五万分之一》《漫长告别》《从未抵达过的地方》《黑猫出现在白天》与文字作品互相解构。音乐发布身份为 K-1647；《五万分之一》专辑从日常生活进入死亡、编辑与协作，并强调作品由多人共同完成。

玩法职责：11 个案件采用 11 个不完整声部或混音空间，不把歌名直接当密码。玩家补齐的不是“正确歌曲”，而是某条故事线被系统滤掉的频段。

## 关系矩阵

| A -> B | 表面关系 | 不对称事实 | 游戏中的冲突 |
|---|---|---|---|
| 斗篷 -> 火爪 | 初识、可能受雇 | 斗篷模仿；火爪用刀测试它 | 遵守火爪规则会加速斗篷获得身份，也可能让玩家失去原身份 |
| 火爪 -> 耳鱼 | 老相识、互相求助 | 火爪怀疑耳鱼因知晓过多而麻木；耳鱼见过她多次死亡 | 同一事故，两人的守则会给出不同生存定义 |
| 耳鱼 -> 客观事实 | 观察者 | 能越过时空却不能修改事实 | 他只能指出可核对项，不能替玩家决定价值 |
| 火爪 -> Husburd | 寻找失踪者 | 思念既驱动她，也限制断/斩 | 玩家不能靠“切断关系”消除这条线 |
| 普森 -> 凯特 | 父女 | 普森的名声使凯特被迫处于注视中 | 所有涉及凯特的档案只写外部行为和他人话语 |
| 和弦 -> 研究员 | 工作系统 | 免费福利、绩效与能源形成隐形交换 | 提交正确答案未必是道德正确答案 |
| K-1647 -> 读者 | 引导者 | 它给名字，也可能写错宾语和坐标 | 系统提示不能自动视为作者真话 |

## 核心循环

每个夜班案件遵循：

`领取密室 -> 阅读三份不同作者的规则 -> 检查日志/声音/色谱 -> 选择遵守对象 -> 操作元器件 -> 决定是否提交答案 -> 角色关系与下一班规则改变`

玩家持有四类永久元器件：

- 光谱旋钮：在 380–700 nm 间调节，显露被颜色覆盖的符号。
- 声谱读带器：分离被火爪能力压住的声音。
- 录音机：保存一个房间的客观记录，跨周目保留。
- 投射卡槽：决定当前行动以谁的身份被系统记录。

## 规则系统

规则必须显示作者或留下可追查的作者痕迹。不存在无来源的“请遵守以下规则”。

### 系统守则

措辞稳定、计量精确、关心提交率，不关心角色是否仍保持原身份。它说的操作事实通常可靠，因果解释不可靠。

### 商徵批注

从实际工作经验出发，关注题型复用、绩效和研究所代价。他可能故意不提交已经解出的题。

### 耳鱼录音

只确认可重复观察的事实。涉及时间顺序时经常停顿，因为他见过多个版本。

### 火爪刻痕

命令短、动作明确，优先保护关系而非系统完整性。她会要求移动、斩断或保留某物。

### 斗篷改写

常改变一句话最后的宾语、代词或主语。不是恶意伪造，而是它理解和成为事物的方式。

## 六个夜班案件

### 00《入职守则》

三版守则都声称自己是最新版。玩家通过打印时间、墨迹和门禁日志确定：版本号不是时间顺序。结尾第一次选择投射身份。

### 01《最后一个宾语》

规则每次刷新都会更换最后一个宾语。玩家需保留刷新前录音，确认斗篷在学习规则，而非系统遭入侵。

### 02《雨停以前》

监控显示死象区域永远下雨，气象日志却记录“晴”。玩家要区分地点规则、观察者恐惧和设备天气字段，不能简单判断一方说谎。

### 03《听不见的第四轨》

火爪的声波压住一轨录音。玩家用声谱读带器找回低频内容，并决定是否把她寻找 Husburd 的私人录音提交给和弦。

这一案同时引入 7461-K：同一低频对人类、非 K-1647 机器人、火爪可能产生三种不同后果。系统先识别实体层，再决定安全规则。

### 04《正确答案》

玩家解开一间 S 级密室，系统承诺大量能源。商徵留下未提交标记。提交会固定一个故事版本，拒绝会触发绩效处分。

### 05《守则的作者》

二周目继承录音机。玩家发现第一班的守则包含自己上一周目的句子。最终选择不是“相信谁”，而是决定自己愿意为哪条规则署名。

## 状态与多周目

不使用善恶值。保存四类状态：

- `recorded`: 玩家亲自保存过的客观记录。
- `submitted`: 交给和弦并被固定为答案的版本。
- `identity`: 当前投射身份及被哪些角色如此称呼。
- `debts`: 玩家从哪位角色处获得元器件，以及尚未回应的请求。

一周目结束只解决“本班守则由谁改写”。二周目解决“为什么系统需要研究员签名”。Grenyes、苍绿之眼本体、凯特最后一案继续保留。

## 文本禁区

- 不写“记忆不是录像”“规则是为了保护你”一类总结句。
- 不让单份报告解释投射、和弦、苍绿之眼全貌。
- 不让耳鱼成为全知解说员。
- 不让火爪只剩武器功能。
- 不把斗篷写成故障病毒。
- 不写凯特内心独白，不代她完成与普森的关系结论。
- 规则失败用日志变化、称呼变化和物件后果呈现，不用死亡弹窗。
- 不因编号相同就把角色、芯片、机器人和作者投影合并。
- 不把机器人四大法律写成可靠道德答案；它是法庭世界中的制度规则。
- 不把真实照护、死亡与告别经历改造成密码或猎奇谜面。
